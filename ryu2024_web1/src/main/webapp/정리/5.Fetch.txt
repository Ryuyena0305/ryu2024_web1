 [ JS 함수 ] : 자바와 다르게 JS함수는 하나의 자료 타입이다.
 	1. 선언적 함수
 		- function 함수명 (매개변수명, 매개변수명){}
 	2. 익명 함수
 		- function (매개변수명,매개변수명){}
 		- const 상수명 = function (매개변수명,매개변수명){}
 	3. 람다식 함수
 		- (매개변수명,매개변수명) => {}
 		- const 상수명 = (매개변수명,매개변수명) => {}
 		
 [ 람다식 활용처 ]
 	1. 배열변수명.forEach( 람다식 )
 		- [ ].forEach(( 반복변수명 ) => { 실행문; })
 		- 배열내 첫번째 요소부터 마지막요소까지 반복변수명 하나씩 대입 반복
 		
	2. let 새로운배열명.map( 람다식 ) 
		- [ ].map(( 반복변수명 ) => { 실행문; })
		- 배열내 첫번째 요소부터 마지막 요소까지 반복함수명에 하나씩 요소를 대입 반복
		- 차이점 : forEach함수는 return 불가능, map 함수는 return 가능하다.
		- map 사용처 : 배열 복사
		
	3. fetch( ).then( 람다식 표현 ).then( 람다식표현 ).catch( 람다식표현 )
